About

Hello! I am LingJun Zhou. Data Scientist, Machine Learning Engineer, and ex-Physicist.

Hello, I'm LingJun Zhou, a data scientist with a Ph.D. in Theoretical Physics and over seven years of experience in machine learning, data science, and software development. My expertise spans the entire data science lifecycle, enabling me to deliver comprehensive, end-to-end solutions.

Technical Proficiencies:

  • Programming Languages: Python, JavaScript/TypeScript, Bash.
  • Tools & Packages
    • Natural Language Processing (NLP): spaCy, Transformers, Haystack
    • Time Series Analysis: Prophet, Sktime, Scikit-learn, XGBoost, LightGBM
    • API server Frameworks: Flask, FastAPI
    • Databases: OpenSearch, SQLite, Redis
    • Infrastructure as Code (IaC): Terraform
    • LLM: Ollama
    • OCR: Tesseract
    • Web scraping: Selenium
    • Containerization: Docker
    • LiDAR data processing: QGIS, LAStools, Open3D, PDAL
    • UI testing: Cypress, Katalon
    • Front-end Frameworks: React JS, Gradio python, Streamlit
    • Others: Hydra (dynamically manage hierarchical configs), Optuna (ML model hyper-parameter tuning), Typer (utility tool)
  • Platforms: AWS, Heroku, Azure, Linux.

At Natural Resources Canada, I have led several impactful projects:

  • Developed a machine learning model for predicting electric and heating demand, significantly improving energy management.
  • Created a flood risk mapping tool using LiDAR point cloud data, enhancing disaster preparedness.
  • Implemented a satellite image segmentation project, advancing remote sensing capabilities.
  • Contributed to the development of large language models (LLMs) and natural language processing (NLP) applications, including named-entity recognition.

I have worked across the full data science lifecycle, including:
  • Assessing AI Readiness and Data Quality
  • Project Scoping and Proposal Development
  • Client and Project Management
  • Deployment and Monitoring
  • ETL, Data Wrangling: Cleaning, Processing, and Feature Engineering
  • Research and Development: Designing and implementing machine learning models
  • MLOps: Integrating DataOps, ModelOps, and DevOps for production-grade solutions
  • End-to-End Pipeline Development: Reducing time from experimentation to production
